Usage: goam [OPTIONS] install

Options:
  -v=false: Verbose
  -d=false: Print debugging messages
  -t=false: Print timings pertaining executed commands
  -dashboard=true: Report public packages at http://godashboard.appspot.com/package

Description:
  Builds and installs files provided by the project, as specified by
  the "GOAM.conf" configuration files of the project.

Command chain:
  goam install <-- goam make <-- goam install-deps
